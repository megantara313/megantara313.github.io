In this project, it was created to fulfill requests for vaccination participants who register online with the aim of accelerating the vaccination registration process held by the provincial government of East Java and the Ministry of Health of the Republic of Indonesia. This online vaccination registration application uses CRUD by storing data for each registrant in a database. This application is made using the CSS framework, namely Bootstrap and PHP5.

in this project, an online registration system was created to meet the request of the East Java provincial government to meet the needs of training institution agencies. In this application prospective participants can register according to personal data on each participant's ID card. After registering participants are required to login to go to the main page. Here The completed participant data will appear along with the chosen training vocational. This website-based application uses CRUD (Cread read update delete). This application is designed using Bootstrap version 5 and PHP version 5.

This project is a final project to obtain a D3 degree from the State Electronics Polytechnic of Surabaya. This dashboard uses esp32 hardware (Internet of think) with c programming language.The hardware of this dashboard is designed using EAGLE software as hardware design software. For casing design using the CAD Inventor application. And for the IDE application the ESP32 program uses Arduino IDE. In this project I was one of the students involved from 3 other students in the collaboration of the Surabaya Electronics Polytechnic with PT. Insera Sena. In that collaboration I played a role in the electric bicycle dashboard development division Even though I was working on a project in the Covid-19 pandemic, in the end, with the guidance of the lecturer, I was able to complete the dashboard on time.
